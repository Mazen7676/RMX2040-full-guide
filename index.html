Realme 6i Guide

In this thread, I'll cover everything you can do with Realme 6i (except unbricking a hard brick).

Rollback to Android 10:

ALL YOUR DATA WILL BE DELETED

Pre-requirements:

  1. Download the downgrade OTA file, you can find it here or type #official_ozip in t.me/narzo10club

Guide:

  1. Open file manager and click the OTA file, it'll automatically rollback.
  2. Alternatively, reboot to recovery, select "Install from storage" and choose the OTA file.

Unlock Bootloader:

ALL YOUR DATA WILL BE DELETED

Pre-requirements:

  1. Install the latest platform tools.
  2. Install the latest drivers, you can find them in androidusbdrivers with guides on how to install.

Guide:

  1. Open Settings > About Phone, click "Build Number" multiple times until prompted for the lock screen password or "You're now in developer mode" message is shown.
  2. Search for "USB debugging" and turn it on.
  3. Install deep testing for RMX2040 (you can find it in t.me/narzo10club or Realme official site).
  4. Click on "Start Applying" (if you don't see a warning screen or application submitted, click "Query Verification Status" then "Start the In-depth Test").
  5. Accept the warning screen and submit the application. Wait for about 1-2 hours, and you should see "Review Successful".
  6. Click "Start the In-depth Test".
  7. You should see a black screen with 3 or more command lines. Connect the phone to the PC.
  8. Open the platform tools folder, click on the address bar where the file path is shown, type `cmd`, and press enter. This should open a command prompt.
  9. Type: fastboot flashing unlock.
  10. Select "UNLOCK THE BOOTLOADER" using the volume buttons, then press the power button.
  11. Redo the first two lines (if you wish to continue with flashing recovery, rooting, and installing ROM).

Debloat:

Use Shizuku + Shizutools or Shizuku + Canta. You can find countless guides online.

Unbrick Soft Brick:

ALL YOUR DATA WILL BE DELETED IF YOU CHOOSE USERDATA IN MCT OFP EXTRACTOR

Pre-requirements:

  1. Install stock firmware. Search "RMX2040 C.12 stock firmware" on your laptop if you're on Android 11, or "RMX2040 A.47 stock firmware" if you're on Android 10. You can find it here.
  2. Install the latest platform tools.
  3. Install the latest drivers, you can find them here with guides on how to install.
  4. Unlocked bootloader.
  5. Download MCT OFP extractor.

Guide:

  1. Boot into bootloader (hold power + volume down). If you're stuck in a bootloop or an infinite loading screen, hold power + vol up + vol down till it turns off, then release volume up.
  2. You should see 3 (or more) command lines on the left. Unzip the stock firmware.
  3. Open MCT OFP Extractor, choose the .ofp file from the unzipped folder, and put it in a new folder (should take some time).
  4. Copy all of the extracted files into the platform tools folder after extracting them.
  5. Connect the phone to the laptop.
  6. Click on the address bar where the file path is shown, type `cmd`, and press enter. This should open a command prompt.
  7. Type the following commands for each file that ends in .img: fastboot flash [img_name] [img_name.img] (for example: fastboot flash boot boot.img).
  8. It's ok if something refuses to flash.
  9. Now type: fastboot reboot system and everything should be back to normal.

Unbrick Hard Brick:

Pre-requirements:

  1. Install stock firmware. Search "RMX2040 C.12 stock firmware" if you're on Android 11, or "RMX2040 A.47 stock firmware" if you're on Android 10. You can find it on the oppostockrom website.
  2. Install the latest drivers, you can find them on the androidusbdrivers website with guides on how to install.
  3. Install MCT OFP extractor.
  4. Install SP Flash Tool.

Guide:

Ask in t.me/narzo10club.

Install Custom Recovery (PBRP):

Pre-requirements:

  1. Rolled back to Android 10.
  2. Unlocked bootloader.
  3. USB debugging enabled.
  4. PBRP for RMX2020 (yes, this is not a typo).

Guide:

  1. Download PBRP for RMX2020 (yes, this is not a typo) from SourceForge.
  2. Copy the entire zip file to your phone, unzip it on your computer, open the TWRP folder in the unzipped folder, copy recovery.img, and paste it into platform tools on your PC.
  3. Remove the password from your mobile (do this whenever trying to open PBRP because decryption is broken).
  4. Connect the phone to the PC, click on the address bar where the file path is shown, type `cmd`, and press enter. This should open a command line.
  5. Enable file transfer on your phone, then type adb reboot bootloader in the cmd. If this doesn't work, try adb reboot fastboot.
  6. It should ask for a prompt on your phone to enable USB debugging. Check "Always allow" and allow, then type the command again.
  7. You should see 3 lines on a black screen on your phone.
  8. Type in the cmd window on your computer fastboot flash recovery recovery.img, then fastboot reboot recovery.
  9. Wait for a bit, you should see PitchBlack recovery loading screen.
  10. Click install, choose the zip you previously copied to your phone, wait for a bit, press the home button, reboot, recovery, to reboot back to the system.

Install Custom Recovery (TWRP):

Pre-requirements:

  1. Rolled back to Android 10.
  2. Unlocked bootloader.
  3. USB debugging enabled.
  4. TWRP for RMX2040 (you can find this in t.me/narzo10club).

Guide:

  1. Download TWRP for RMX2040 (you can find this in t.me/narzo10club).
  2. Unzip it on your computer, copy recovery.img, and paste it into the platform tools on your PC.
  3. Connect the phone to the PC, click on the address bar where the file path is shown, type `cmd`, and press enter. This should open a command line.
  4. Enable file transfer on your phone, then type adb reboot bootloader in the cmd. If this doesn't work, try adb reboot fastboot.
  5. It should ask for a prompt on your phone to enable USB debugging. Check "Always allow" and allow, then type the command again.
  6. You should see 3 lines on a black screen on your phone.
  7. Type in the cmd window on your computer fastboot flash recovery recovery.img, then fastboot reboot recovery.
  8. Wait for a bit, you should see TWRP loading screen.
  9. To reboot back to the system, click reboot-system.

Root:

Pre-requirements:

  1. Custom recovery installed.
  2. Magisk.

Guide:

  1. Flash Magisk normally via PBRP.
  2. Find countless guides online on how to bypass SafetyNet check and hide root via Shamiko.

Install ROM:

ALL YOUR DATA (might) BE DELETED

Pre-requirements:

  1. TWRP installed.
  2. GApps or FOSSApps of your preference if not already included in the GSI (optional).
  3. A ROM of your choice on your PC (you can find RMX2020 ROMs that work for RMX2040 in t.me/rmx2040roms). (I personally tried crDroid 12.1 ROM NOT GSI and it's working well so far, will update on further testing.)

Guide:

  1. Connect the phone to the PC, open platform tools, click on the address bar where the file path is shown, type `cmd`, and press enter. This should open a command line.
  2. Enable file transfer on your phone, then transfer GApps (optional) and PBRP zip files from the PC to your phone.
  3. Type adb reboot recovery in the command prompt. You should see the TWRP loading screen (PBRP fastboot is broken).
  4. Click "Wipe" > "Advanced Wipe", select data, cache, Dalvik/ART cache, and swipe to wipe.
  5. Click the home button, then "Reboot" > "Fastboot". You should be in TWRP fastbootd.
  6. Type the following commands: fastboot erase metadata, fastboot erase system, and fastboot reboot recovery.
  7. In TWRP, click "Install", go to PBRP zip file location, click it, swipe to install, then click the home button, "Reboot" > "Recovery".
  8. In PBRP, click "Advanced" > "ADB Sideload", check "Wipe Dalvik Cache" and "Wipe Cache", then swipe.
  9. Type adb sideload [zip-file-name-of-installed-os.zip] (don't put the square brackets). It should take some time. If it fails on 48%, check the phone for success.
  10. Click "Cancel", then "Home" > "Wipe" > "Format Data", type "yes", and click enter.
  11. Click "Home", then "Reboot" > "System".

Installing GApps or FOSSApps (optional):

  1. In PBRP recovery, go to "Install", find and click the GApps or FOSSApps zip file, then swipe to install.
  2. Go to "Wipe" > "Format Data", type "yes", and enter.
  3. Finally, go to "Reboot" > "System".

Install GSI (Not Recommended as it Will Most Likely Lag):

ALL YOUR DATA WILL BE DELETED

Pre-requirements:

  1. TWRP installed.
  2. vbmeta.img file (extract it from stock firmware using MCT OFP extractor or get it from t.me/rmx2040roms).
  3. GApps or FOSSApps of your preference if not already included in the GSI (optional).
  4. A GSI of your choice on your computer (must be arm64 and not vndklite, with the code starting with b: bvN or bgN or bvS or bgS). (I personally like dotOS from their official site or AOSP 14 if you like it).

Guide:

  1. Extract the zip or xz file using 7-Zip or WinRAR.
  2. Connect your phone to the PC, open platform tools, click on the address bar where the file path is shown, type `cmd`, and press enter. This should open a command prompt window.
  3. Type adb reboot fastboot in the command prompt. You should see TWRP fastbootd.
  4. Type the following commands: fastboot --disable-verification flash vbmeta vbmeta.img, fastboot reboot fastboot, fastboot erase system, fastboot flash system [name-of-gsi-img.img] (don't put the square brackets), fastboot -w, fastboot reboot bootloader, fastboot erase userdata, fastboot reboot system.

Installing GApps or FOSSApps (optional):

  1. Transfer the zip file from the PC to your phone.
  2. Type adb reboot recovery in TWRP recovery.
  3. Go to "Wipe" and swipe.
  4. Go to "Install", find and click the GApps or FOSSApps zip file, then swipe to install.
  5. Go to "Wipe" > "Format Data", type "yes", and enter.
  6. Finally, go to "Reboot" > "System".
